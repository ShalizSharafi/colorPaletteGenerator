## ✨ Features

### Palette Generator
- Generates random RGB color palettes on page load
- **Regenerate** the entire palette with one click
- **Add up to 5 new color boxes** to the palette using the hover `+` button or the navbar button
- **Refresh** individual colors without affecting the rest
- **Lock** a color to protect it from being changed
- **Delete** color boxes (up to 4 deletions)
- **Copy** any color code to clipboard with one click
- **Save** your current palette to the view board

### Trending Section
- Browse a curated grid of random hex color palettes
- Click any color in a card to paste its hex code into the card label
- **Add up to 6 new color cards** to the trending section
- **Delete** cards you no longer need
- Counter tracks how many cards have been added

### View Board
- Slide-in panel to view all saved palettes
- Open and close with smooth animation

### UI & UX
- Responsive design — works on mobile, tablet, and desktop
- Hover interactions reveal action buttons (refresh, lock, delete, copy)
- Ad popup with Go Pro modal
- Clean footer with profile, tools list, and palette of the day

## 🚀 Getting Started

Clone the repo and open `index.html` in your browser:

```bash
git clone https://github.com/shalizsharafi/colorPaletteGenerator.git
cd colorPaletteGenerator
open index.html
```

No build steps or dependencies required!
